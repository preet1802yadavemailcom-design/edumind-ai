## 🚀 Quick Start (3 Steps)

```bash
# 1. Install
npm install

# 2. Add your FREE Groq API key
cp .env.example .env
# Edit .env → paste VITE_GROQ_API_KEY=gsk_your_key

# 3. Launch
npm run dev
# Open: http://localhost:5173
```

### Get FREE Groq API Key
1. Go to **https://console.groq.com**
2. Sign up → API Keys → Create Key
3. Copy key (starts with `gsk_`) → paste in `.env`

## 🔧 Common Issues

**API not working?**
```bash
# Check .env file
cat .env  # Should show: VITE_GROQ_API_KEY=gsk_xxx

# Restart server after adding key
npm run dev
```

**Port in use?**
```bash
npm run dev -- --port 3000
```

**Clean install:**
```bash
rm -rf node_modules package-lock.json
npm install
npm run dev
```
